# El Bote

Las cuentas de la peña, sin líos. Apuntad lo que trae cada uno, quién lo paga,
y El Bote os dice quién le debe cuánto a quién para quedar en paz.

Es una sola página (`index.html`), sin servidor ni base de datos: todo se
guarda en el propio navegador. Ábrela y listo.

**Pruébalo aquí:** https://adpablos.github.io/el-bote/

## Desarrollo

No hay build ni dependencias. Abre `index.html` en el navegador o sirve la
carpeta con cualquier servidor estático:

```sh
python3 -m http.server
```
